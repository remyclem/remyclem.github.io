Quelques commandes Linux

Compression / Décompression avec tar

  • Compression : 
    tar -czvf archive.tar.gz /chemin/vers/element
  • Décompression : 
    tar -xzvf archive.tar.gz

Utilisation de screen

  • Créer une session : 
    screen -S nom_de_session
  • Détacher une session : 
    Appuyer sur Ctrl + A + D
  • Lister les sessions : 
    screen -list
  • Attacher une session : 
    screen -r nom_de_session
  • Tuer une session : 
    screen -X -S nom_de_session kill

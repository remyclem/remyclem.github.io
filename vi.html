Commandes de base avec VI

Au secours sortez-moi de là !

Pour quitter VI sans risquer de sauvegarder le fichier :
  • Appuyer sur Echap
  • Taper :q! puis Entrée

Edition de texte

  • Ouvrir le fichier
  • Passer en mode édition
  • Sauvegarder
  • Quitter

Explication sur les Modes de VI

VI dispose de plusieurs modes :
  • Commande, qui permet de se déplacer dans le fichier.
    Le déplacement peut se faire à l'aide des touches directionnelles ou des touches h, j, k, l. Dans ce mode il est possible d'effectuer des actions telles que sauvegarder ou bien quitter.
  • Edition
Au démarrage, VI est en mode commande.
